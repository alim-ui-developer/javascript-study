filter 문제

  • 이름에 a를 포함한 사람들을 출력하시오.
  • 이름에 같은 글자가 연속해서 들어간 사람을 출력하시오. (예-tt,ff,ll 이런 글자들)

정답

1. 이름에 a를 포함한 사람들을 출력하시오.

let namesNewArray = [];

names.filter((name) => {
  return name.includes('a') && namesNewArray.push(name);
})

2. 이름에 같은 글자가 연속해서 들어간 사람을 출력하시오. (예-tt,ff,ll 이런 글자들)

let namesNewArray = [];

names.filter((name) => {
  let nameAlphabets = [...name.replace(/\s/g, "")];

  for(let i = 0; i < nameAlphabets.length; i++){
    nameAlphabets[i] === nameAlphabets[i-1] && !namesNewArray.includes(name) && namesNewArray.push(name);
  }
  return namesNewArray;
})

some 문제

  • 전체 이름의 길이가 20자 이상인 사람이 있는가?
  • 성을 제외한 이름에 p를 포함한 사람이 있는가?(대소문자 상관 no)

정답

1. 전체 이름의 길이가 20자 이상인 사람이 있는가?

const question1 = names.some((name) => {
  return name.length >= 20;
})

console.log(question1);

2. 성을 제외한 이름에 p를 포함한 사람이 있는가?(대소문자 상관 no)

const question2 = names.some((name) => {
  const getName = name.split(" ")[0];
  return getName.toLowerCase().includes('p');
})

console.log(question2);

every 문제

  • 모두의 전체 이름의 길이가 20자 이상인가?
  • 모두의 이름에 a 가 포함되어 있는가?

정답

1. 모두의 전체 이름의 길이가 20자 이상인가?

const question1 = names.every((name) => {
  return name.length >= 20;
})

console.log(question1);

2. 모두의 이름에 a 가 포함되어 있는가?

const question2 = names.every((name) => {
  return name.toLowerCase().includes('a');
})

console.log(question2);

find 문제

  • 전체 이름의 길이가 20자 이상인 사람을 찾으시오.
  • 미들네임이 포함되어있는 사람을 찾으시오.(예-Steven Paul Jobs)

정답

1. 전체 이름의 길이가 20자 이상인 사람을 찾으시오.

let namesNewArray = [];

names.filter((name) => {
  return name.length >= 20 && namesNewArray.push(name);
})

2. 미들네임이 포함되어있는 사람을 찾으시오.(예-Steven Paul Jobs)

let namesNewArray = [];

names.filter((name) => {
  const nameLength = name.split(' ').length;
  return nameLength >= 3 && namesNewArray.push(name);
})

findIndex 문제

  • 전체 이름의 길이가 20자 이상인 사람의 인덱스 번호를 찾으시오.
  • 미들네임이 포함되어있는 사람의 인덱스 번호를 찾으시오.

정답

1. 전체 이름의 길이가 20자 이상인 사람의 인덱스 번호를 찾으시오.

let namesIndexArray = [];

const getIndex = () => {
  names.findIndex((name, index) => {
    name.length >= 20 && namesIndexArray.push(index);
  })

  return namesIndexArray;
};

getIndex();

2. 미들네임이 포함되어있는 사람의 인덱스 번호를 찾으시오.

const getIndex = () => {
  names.findIndex((name, index) => {
    const nameLength = name.split(' ').length;
    nameLength >= 3 && namesIndexArray.push(index);
  })

  return namesIndexArray;
}

getIndex();
